# Notebook

LaTeXで作成するドット方眼ノートページジェネレーター

## 概要

このプロジェクトは、印刷可能なドット方眼ノートページをPDFで生成するためのLaTeXテンプレート集です。A4サイズとバイブルサイズの2種類の用紙サイズに対応しています。

## ファイル構成

- `notebook.cls` - カスタムLaTeXクラスファイル
  - A4用紙とバイブルサイズ用紙のレイアウトを定義
  - マージン、ヘッダー、フッターの設定を含む
- `a4_grid.tex` - A4サイズのドット方眼ページ
- `bible_grid.tex` - バイブルサイズ（95mm×171mm）のドット方眼ページ
- `Makefile` - ビルド自動化ファイル

## 必要な環境

- pLaTeX (日本語LaTeX環境)
- dvipdfmx
- TikZ パッケージ

## 使い方

### すべてのPDFを生成

```bash
make
```

または

```bash
make pdf
```

### 個別に生成

```bash
make a4_grid.pdf
```

```bash
make bible_grid.pdf
```

### PDFをプレビュー

```bash
make preview
```

※ プレビューコマンドは環境に応じて自動的に選択されます（macOS: Preview、Linux: evince/acroread、Windows: cygstart）

### 生成ファイルをクリーンアップ

```bash
make clean
```

## グリッド仕様

### A4サイズ
- 用紙サイズ: A4 (210mm×297mm)
- ドット間隔: 5mm
- 2ページ分のグリッドを生成

### バイブルサイズ
- 用紙サイズ: 95mm×171mm
- ドット間隔: 5mm
- 2ページ分のグリッドを生成

## カスタマイズ

各TeXファイルを編集することで、以下のカスタマイズが可能です：

- グリッドの間隔（`step`パラメータ）
- ページ数
- ヘッダーの内容（`\rhead`コマンド）
- グリッドの色とスタイル

## ライセンス

このプロジェクトは自由に使用・改変できます。
